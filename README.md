# MeteoLabZPelletier
**Logiciel d'interprétation de données météo**
---
MétéoLab permet à l'utilisateur d'entrer et d'importer des données météo tel que la température, les précipitations, l'humidité, etc.
Il est aussi possible d'interpréter des données météo sur un intervalle de temps choisi par l'utilisateur. Il est possible de calculer
la moyenne, le min-max ainsi que l'écart-type de chaque donnée.
---
Le "Tech Stack":
>Language : C#
>Framework : Uno Platform
>UI Markup : XAML.
>---
